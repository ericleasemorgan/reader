Distant Reader Study Carrels for Project CORD

The result of the Distant Reader process is the creation of a "study carrel", and below is a directory some such things. More specifically, all of these study carrels were created from subsets of a larger data set -- CORD-19, a collection of scientific journal articles on the topic of COVID-19.
